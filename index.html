DevOps Tools and Technologies

Linux

The foundation of DevOps. Learn the basics of Linux commands, shell scripting, and system administration.

Docker

Containerization technology that allows you to package applications with all dependencies into a standardized unit.

Kubernetes

An orchestration platform for managing containerized applications across multiple hosts.

AWS

A cloud computing platform offering services like computing power, storage, and databases on demand.

Jenkins

A continuous integration and continuous delivery (CI/CD) tool that automates the build, test, and deployment process.

Git & GitHub

Version control systems for tracking changes in code and collaborating with other developers.

Terraform

An infrastructure-as-code tool that allows you to define and provision data center infrastructure using a declarative configuration language.

OpenShift

A Kubernetes-based platform for developing and managing containerized applications.

Prometheus & Grafana

Monitoring and alerting tools that help you visualize and analyze the performance of your applications and infrastructure.
